百钱买百鸡

现有公鸡 A 元一只， 母鸡 B 元一只， 小鸡 C 元一只。
林克手上共有 N 元现金，你能告诉他怎么买可以正好全部花完手上的现金？

in

输入三个整数 A、B、C ( 0 < A、B、C < 100 ) 分别代表公鸡、母鸡、小鸡的价格 。
接下来输入一个整数 N ( 0 < A、B、C < 1000 ) 代表林克手上的现金。

out

输出所有满足可能的组合。
如果有多种可能性，按照公鸡的数量从小到大排序，公鸡数量相同的情况下按照母鸡的数量从小到大排序。

Sample

in

12 14 16
100

out

0 6 1
1 4 2
2 2 3
3 0 4
6 2 0
7 0 1

Hint
